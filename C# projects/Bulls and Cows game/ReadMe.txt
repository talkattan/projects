The game “Bulls and Cows"

The first form will enable the user to choose the desired number of rows in the board:each click on the button "number of chances" will increase the counter by one (minimum – 4, maximum – 10and so on, in a loop).

When the user clicks the arrow button to seal the guess, the feedback is displayed in the set of 4 little buttons on the right :black for a precise hit, yellow for a misplaced color
